## 3. 机票价格预测算法设计

模型选择：Regression Tree
神经网络算法，树形算法，决策树，回归决策树

决策树：回归树，CART，GINI指数，交叉验证，RMSE

可选features
- 航空公司
- 出发日期（月、日、周几、小时）
- 到达小时
- 距出发日多少天
- 原价（折扣）

实现：PySpark，MlLib
  
### 3.1 价格（折扣）变化曲线（北京-昆明）

- 数据：2019年北京至昆明每一航班出发前45天的价格信息
- 特征：航空公司、出发时间、到达时间、提前购买天数
- 标签：机票价格、折扣
- 算法：决策树回归，依据：那篇论文，误差：RMSE

### 3.2 最低价格（北京-昆明）

- 特征：航空公司、出发时间、到达时间
- 标签：出发前45天内最低价格
- 算法：决策树回归
